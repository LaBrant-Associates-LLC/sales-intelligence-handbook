## QC-001 Flooring Addendum Fields

### F1. Flooring Work Summary (Daily)

| Field | Input |
|-------|-------|
| Flooring Scope Active Today? | Yes/No |
| Area Installed Today | SF / LF |
| Material Type | LVP Click / LVT Glue / Carpet / Base / Other |
| Rooms / Units / Areas | Text |
| Installer Names | Text |
| Start Time / Stop Time | Time |

### F2. Environmental & Readiness Confirmation (Daily)

| Field | Input |
|-------|-------|
| HVAC Operational? | Yes/No |
| Space Conditioned ≥48 Hours? | Yes/No |
| Ambient Temp (AM / PM) | °F |
| Relative Humidity (AM / PM) | % |
| Deviations from Spec? | Yes/No + explanation |

**STOP-WORK TRIGGER:** HVAC off or conditions out of range.

### F3. Substrate & Moisture Status

| Field | Input |
|-------|-------|
| Moisture Testing Required Today? | Yes/No |
| ASTM Method | F2170 / F1869 / Other |
| Latest Readings | Numeric |
| Within Manufacturer Limits? | Yes/No |
| GC Written Substrate Acceptance on File? | Yes/No |

**If acceptance = NO → INSTALLATION PROHIBITED.**

### F4. Material Control & Dye Lot Tracking

| Field | Input |
|-------|-------|
| Manufacturer / Product | Text |
| Dye Lot / Batch Number | Text |
| Single Lot Verified for Area? | Yes/No |
| Attic Stock Identified & Tagged? | Yes/No |

### F5. Protection & Damage Control (End of Day)

| Field | Input |
|-------|-------|
| Floor Protection Installed? | Yes/No |
| Protection Type | Ram Board / Rosin / Other |
| Other-Trade Damage Observed? | Yes/No → QC-004/QC-005 |

## Required Photo Sets

### Photo Set A — Pre-Install

- Substrate condition (wide shot)
- Flatness check (straightedge visible)
- Moisture test setup + reading
- HVAC running / thermostat

### Photo Set B — In-Progress (Daily)

- Active installation
- Packaging showing dye lot
- Adhesive spread or click-lock engagement

### Photo Set C — End of Day

- Completed area
- Protection installed
- Transitions / edges (close-up)

### Photo Set D — Exceptions

- Any deviation, damage, or stop-work condition

### Photo Naming Convention

`[JobNumber]_[Date]_[PhotoSet][Sequence]_[Description].jpg`

Example: `2025-001_20250115_A01_SubstrateWideShot.jpg`

## First Floor Gate (QA-008) Flooring Criteria

When flooring scope is active, QA-008 v2.1 includes flooring-specific criteria.

### All Must PASS

- Moisture tests compliant
- Substrate acceptance documented
- Environmental logs clean
- Dye lot control verified
- Acclimation documentation complete
- Zero undocumented scope deviations

**Any FAIL = STOP & DIAGNOSE (Owner approval required to continue).**

---

## Superintendent Daily Log Integration

### Reconciliation Rules (Non-Negotiable)

- If Units/Area Installed in Unit Log > 0, a QC-001 entry **must** exist for the same date
- If Acceptance = NO in Unit Log, Units Installed **must equal zero**
- Any Damage Observed requires NCR issuance same day

### Daily Checklist

- ☐ Flooring fields completed (if active)
- ☐ QC-001 submitted with photos
- ☐ Environmental snapshot matches QC-002
- ☐ Acceptance verified before install
- ☐ Protection installed at EOD
