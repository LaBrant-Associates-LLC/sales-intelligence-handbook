# Chapter 1: Operating System

**Purpose:** Daily operating cadence and weekly rhythm for sales execution.

---

## Weekly Cadence

| Day | Activity | Output |
|-----|----------|--------|
| Monday | Pipeline review -- scan VendorNet, DemandStar, QuestCDN, Bonfire | Updated bid list |
| Tuesday | GC outreach -- 3 calls minimum to [Strategic Partners](ch3-gc-intelligence/strategic-partners.md) | Call log in CRM |
| Wednesday | Estimating focus -- no new outreach | Proposals submitted |
| Thursday | Follow-up -- chase pending proposals >7 days old | Status updates in CRM |
| Friday | Admin -- update [CRM](ch8-sales-system.md), prep Monday pipeline | Clean pipeline |

---

## Definition of Done (Outreach)

An outreach is complete when:

1. Contact made (phone or email)
2. Logged in Zoho CRM with date and notes
3. Next action scheduled with specific date
4. If no response after 3 attempts, move to "Dormant" status
